指標背離運動

-------------------------------------------------------------

(1)價格與指標背離狀態<br>
(2)價格與成交量背離狀態 (量價背離)<br>
(3)價格對價格(期貨、現貨) 價格不同步(前者突破新高、後者沒有)<br>
(4)期貨對選擇權(期貨價格創高創低、遠則全是否也創高創低)<br>

(1)可使用RSI指標背離現象
基本策略之下，如何使用RSI背離現象當作濾網，來當作策略?


04:00 使用60分K-BAR線，因為如果是用日線，背離現象比較難抓。<br>
06:20 策略說明 破低買進 10根後出場 無停損。<br>
08:10 績效表。<br>
